# Data Folder

## Included

- `sample_data/`: small `.npz` bundles for quick checks
- `processed_schema.md`: expected processed split schema
- `splits/`: target location for full processed MB1 / MB2 splits

## Not Included

The full raw Alibaba dataset is not bundled here. Download instructions are in:

- `../DATA_ACCESS.md`

```text
data/splits/MB1_round1/
data/splits/MB1_round2/
data/splits/MB1_round3/
data/splits/MB2_round1/
data/splits/MB2_round2/
data/splits/MB2_round3/
```
